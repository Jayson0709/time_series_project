Required Libraries/Modules

To run this project, several dependencies should be installed. They are all list in the requirements.txt file.

Under the project directory, simply run:
pip3 install -r requirements.txt

Running the Project

To run the whole project, simply run the following command under the project directory.
python3 server.py

Then visit http://127.0.0.1:5000/ in your browser. The app should be up & running.
